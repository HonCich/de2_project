Detailed Description

Implementation file for the MQ135 gas sensor library.

Author
G.Krocker (Mad Frog Labs), modifications by P. Horsky @license GNU GPLv3

First version of an Arduino Library for the MQ135 gas sensor TODO: Review the correction factor calculation. This currently relies on the datasheet but the information there seems to be wrong.
